Detailed Description

Singleton class that allows getting data from a single Azure Kinect instance and that is used by AzureKinectColorCamera and AzureKinectDepthCamera.

The method UpdateCapture() updates the capture object if UpdateCapture() was already called with the same id before. If UpdateCapture() was not yet called by the id, the same capture is used. If the capture is updated, all memory values except for the id that called the function are reset. All methods that are required to operate multiple Camera objects are thread-safe.
