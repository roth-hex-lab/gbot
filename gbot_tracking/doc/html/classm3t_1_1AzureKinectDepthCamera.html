Detailed Description

Camera that allows getting depth images from an AzureKinect camera.

Parameters
image_scalescales image to avoid borders after rectification.
use_color_as_world_framespecifies the color camera frame as world frame and automatically define camera2world_pose as depth2color_pose.
depth_offsetconstant offset that is added to depth image values and which is provided in meter.
