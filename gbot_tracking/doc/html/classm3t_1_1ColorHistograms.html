Detailed Description

Class that computes and holds a color histogram for foreground and background and that is used by the RegionModality.

It allows adding pixel colors to the internal memory using AddForegroundColor() and AddBackgroundColor(). Based on those memory values, the foreground and background color histograms can be calculated using InitializeHistograms() and UpdateHistograms(). UpdateHistograms() thereby uses online adaptation with the defined learning rates to take into account previous histogram values. To get the probabilities for a specific pixel_color, the method GetProbabilities() is used. The internal memory can be cleared using ClearMemory().

Parameters
n_binsnumber of bins that is used to discretize each dimension of the RGB color space. Has to be 2, 4, 8, 16, 32, or 64.
learning_rate_flearning rate that is used in the update of the foreground histogram.
learning_rate_blearning rate that is used in the update of the background histogram.
