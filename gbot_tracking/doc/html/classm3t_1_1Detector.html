Detailed Description

Abstract class that is able to set the poses of Link and Body objects assigned to a referenced Optimizer objects.

Using the main method DetectPoses(), the poses of all Link and Body objects of Optimizer objects that are specified in names are defined. Names of optimizers are returnd in detected_names.

Parameters
reset_joint_posesreset the joint poses of all referenced Link objects.
