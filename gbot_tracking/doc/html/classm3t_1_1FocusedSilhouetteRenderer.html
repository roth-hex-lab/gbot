Detailed Description

Renderer that extends the focused depth renderer class with functionality from SilhouetteRendererCore to render images with a defined size that is focused on referenced bodies.

It is able to render both a depth image and a silhouette image where the silhouette_id of Body objects is used as intensity value for pixels on the silhouette. Rendering is started using StartRendering(). Images are fetched from the GPU using FetchSilhouetteImage() and FetchDepthImage(). They can then be accessed using the silhouette_image() and depth_image() getter. Setters and all main methods are thread-safe.

Parameters
id_typetype of ID from Body object that is rendered. BODY = 0, REGION = 1.
