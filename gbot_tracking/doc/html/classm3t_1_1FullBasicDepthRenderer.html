Detailed Description

Renderer that extends the full depth renderer class with functionality from BasicDepthRendererCore to render a depth image.

Rendering is started using StartRendering(). Images are fetched from the GPU using FetchDepthImage() and can then be accessed using the depth_image() getter. Setters and all main methods are thread-safe.
