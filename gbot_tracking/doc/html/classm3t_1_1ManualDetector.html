Detailed Description

Detector that allows a user to select 4 points in the image and that, based on those points, infers the pose of the root_link and updates all Link and Body objects referenced by the Optimizer object.

To select points, the PointDetector class is employed. Using the main method DetectPoses(), the detection process is started and poses of referenced Link and Body objects are set.

Parameters
optimizer_ptrreferenced Optimizer object for which referenced body and link poses are set.
color_camera_ptrreferenced ColorCamera object that is used.
reference_points4 3D model points on the body's surface that are given in the body reference frame.
detector_image_pathoptional path to a helper image in which the position of reference_points can be marked.
