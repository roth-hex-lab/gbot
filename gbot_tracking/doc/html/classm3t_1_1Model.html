Detailed Description

Abstract class that precomputes and stores geometric information from Body objects that is required by Modality objects during tracking.

Typically, the class creates a Sparse Viewpoint Model of the Body, which stores relevant information for all possible viewpoints from virtual cameras around the object.

Parameters
body_ptrreferenced Body object for which the model is generated.
model_pathpath that specifies the location of the model file. If the model file does not exist or different parameters were used, it is generated again. Using INFER_FROM_NAME in the metafile sets the path to <name>.bin.
sphere_radiusdistance from the object center to the center of a virtual camera.
n_dividesnumber of times an icosahedron is divided to generate viewpoints on a sphere. It thereby controls the number of template views.
n_pointsnumber of points that are sampled for each view.
max_radius_depth_offsetmaximum radius in meter for the calculation of depth_offsets that are used for occlusion handling.
stride_depth_offsetdistance between points in meter that are used to calculate depth_offsets.
use_random_seedif a random seed is used to sample points.
image_sizesize of images that are rendered for each view.
