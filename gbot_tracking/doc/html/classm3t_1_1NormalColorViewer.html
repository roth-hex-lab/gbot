Detailed Description

Viewer that overlays color images from a ColorCamera with normal renderings based on the geometry stored in the RendererGeometry.

Parameters
renderer_geometry_ptrreferenced RendererGeometry object that defines the geometry that is displayed.
opacitydefines the transparency of pixels from the normal rendering.
